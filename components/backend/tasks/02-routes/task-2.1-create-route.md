## Описание
Создавать персонализированный маршрут по введенным параметрам поездки (локация, даты, интересы) с использованием AI-модуля.

## Требования
- Принимать место поездки, даты и интересы.
- Вызывать AI-сервис генерации маршрута.
- Сохранять маршрут в БД.
- Возвращать структуру, пригодную для отображения на карте и в списке.

## Технические детали
- Основные файлы: `components/backend/services/gemini_handler.py`, `components/backend/main.py`, `components/backend/schemas.py`.
- Контракт: `POST /routes`.
- Обработать ошибки внешнего AI API и вернуть понятное сообщение.

## Критерии приемки
- [ ] Маршрут генерируется для валидного запроса.
- [ ] Результат сохраняется в БД.
- [ ] Формат ответа соответствует фронтенду.
- [ ] Ошибки AI/API корректно обрабатываются.
