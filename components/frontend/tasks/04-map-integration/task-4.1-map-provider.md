## Описание
Интегрировать карту в интерфейс маршрута для отображения точек и контекста поездки без привязки к конкретному внешнему провайдеру маршрутов.

## Требования
- Показывать карту в экране маршрута.
- Поддерживать базовые действия: масштабирование, центрирование.
- Корректно обрабатывать ошибки загрузки карты/API.
- Не зависеть от удаленной логики построения маршрута.

## Технические детали
- Основные файлы: `components/frontend/src/components/YandexMap.tsx`, `components/frontend/src/components/ChatFrame.tsx`.
- Использовать ключ API из переменных окружения.
- Stub-flow маршрута должен оставаться локальным.

## Критерии приемки
- [ ] Карта загружается в основном сценарии.
- [ ] Ошибки сети/ключа не ломают страницу.
- [ ] Проверено на desktop и mobile.
- [ ] Кнопка генерации маршрута не вызывает внешний route API.
