## Информация
- **Status**: IN PROGRESS

- **Use Case**: UC-1

## Описание
Сделать страницу регистрации с клиентской валидацией и обработкой ошибок API.

## Требования
- Поля: email, пароль, подтверждение пароля.
- Валидация формата и обязательных полей.
- Понятные сообщения об ошибках.
- Успешная регистрация переводит пользователя к следующему шагу (вход/главная).

## Технические детали
- Основные файлы: `components/frontend/src/components/Signup.tsx`, `components/frontend/src/pages/Index.tsx`.
- Интеграция с backend endpoint `POST /register`.
- Состояния: idle/loading/success/error.

## Критерии приемки
- [ ] Форма отправляет валидные данные.
- [ ] Ошибки backend отображаются пользователю.
- [ ] UI корректно работает на мобильных и desktop.
- [ ] Логика покрыта минимум базовыми тестами/чек-листом.
