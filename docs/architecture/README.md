# Architecture Docs

- UML (text): `docs/architecture/uml-text.md`
- API contracts: `docs/contracts/api-contracts.md`
- Service contracts: `docs/contracts/service-contracts.md`

## Scope
Документы отражают текущее состояние backend и интеграций по коду в `components/backend/*`.

## Update Rules
1. Добавили/изменили endpoint -> обновить `api-contracts.md`.
2. Изменили orchestration между service/repository -> обновить `service-contracts.md`.
3. Изменили архитектурные границы -> обновить диаграммы в `uml-text.md`.
